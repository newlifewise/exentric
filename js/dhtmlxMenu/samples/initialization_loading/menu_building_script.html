Sample: dhtmlxMenu Building Menu with script dhtmlxMenu main page

You can build menu without xml loading


 
<div id="menu_zone" style="width:600; height:50;"/>
<script>
 
        //init menu
        aMenuBar=new dhtmlXMenuBarObject('menu_zone','100%',30,"Demo menu");
        // set processing function
        
        //create menu item
        var item = new dhtmlXMenuItemObject("file_id","File","");
        aMenuBar.addItem(aMenuBar,item);
        //create submenu
        var subMenu = new dhtmlXMenuBarPanelObject(aMenuBar,item,false,120,true);        
        //add item to submenu
        var item = new dhtmlXMenuItemObject("exit_id","Exit","");
        aMenuBar.addItem(subMenu,item);
                
        //create menu item
        var item = new dhtmlXMenuItemObject("help_id","Help","");
        aMenuBar.addItem(aMenuBar,item);
        //create submenu
        var subMenu = new dhtmlXMenuBarPanelObject(aMenuBar,item,false,120,true);                
        //create menu item
        var item = new dhtmlXMenuItemObject("index_id","Index","");        
        aMenuBar.addItem(subMenu,item);                        
        //create divider
        var subMenu = new dhtmlXMenuDividerYObject("id_0");                
        aMenuBar.addItem(subMenu,item);                        
        //create menu item
        var item = new dhtmlXMenuItemObject("about_id","About","");
        aMenuBar.addItem(subMenu,item);
</script>
Parameters passed to the constructor are:
  • object to attach menu to (should be loaded before calling constructor)
  • width of the menu
  • height of the menu
  • title of menu
